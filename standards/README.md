# **Engineering Standards**

This directory contains all Engineering Standards for the organization. Standards are living documents that define current best practices, required patterns, and operational rules.

### **Purpose of Standards**
- Define "how we do things here."
- Ensure consistency and alignment across teams.
- Provide clear guidance for engineering practices.

### **How to Use This Directory**
1. **Browse Standards**: Use the [Standards Catalog](catalog.md) to find specific standards.
2. **Propose Changes**: Submit an RFC or update an existing standard as needed.
3. **Follow Conventions**: Ensure new standards follow the [Standards Template](template.md).

### **Key Files**
- [Standards Catalog](catalog.md): Overview of all standards.
- [Standards Template](template.md): Template for creating new standards.
