# **Standards Committee**

This directory contains all documentation related to the Standards Committee. The Standards Committee is responsible for maintaining and updating engineering standards to ensure consistency and alignment across the organization.

### **Purpose of the Standards Committee**
- Maintain and update engineering standards.
- Ensure consistency across domains.
- Review proposed changes to standards.
