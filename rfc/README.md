# **Requests for Comment (RFCs)**

This directory contains all Requests for Comment (RFCs) for the organization. RFCs are proposals used to gather feedback, alignment, and early visibility before committing to a direction.

### **Purpose of RFCs**
- Explore options and gather input.
- Socialize ideas before significant investment.
- Propose changes that may lead to ADRs or Standards.

### **How to Use This Directory**
1. **Browse RFCs**: Use the [RFC Catalog](catalog.md) to find specific RFCs.
2. **Propose Changes**: Submit a new RFC using the [RFC Template](template.md).
3. **Track Status**: Check the status of RFCs in the catalog (e.g., Draft, Accepted, Rejected).
4. **Review SLAs**: Refer to the [SLA Document](../governance/sla.md) for timeframes related to RFC reviews.

### **Key Files**
- [RFC Catalog](catalog.md): Overview of all RFCs.
- [RFC Template](template.md): Template for creating new RFCs.
